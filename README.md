# Streak 🔥

Build habits that stick. Streak helps you track your daily habits, visualize your consistency, and keep your streak alive — one day at a time.

## Features

- **Track habits** — Create and manage your daily habits
- **Mark completions** — Check off habits as you complete them each day
- **Streak counter** — See your current streak for each habit
- **History** — View your completion history over time
- **Daily reminders** — Set per-habit notifications so you never miss a day

## Tech

Built with [Expo](https://expo.dev) and [React Native](https://reactnative.dev).

- **Navigation** — [Expo Router](https://expo.github.io/router) (file-based)
- **Storage** — [expo-sqlite](https://docs.expo.dev/versions/latest/sdk/sqlite/)
- **Notifications** — [expo-notifications](https://docs.expo.dev/versions/latest/sdk/notifications/)
- **Animations** — [react-native-reanimated](https://docs.swmansion.com/react-native-reanimated/)

## Getting started

```bash
npm install
npx expo start
```

Press `a` to open in the Android emulator, or scan the QR code with [Expo Go](https://expo.dev/go) on your phone.
